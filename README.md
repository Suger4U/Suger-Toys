A personal collection of small web toys, mini games, experimental projects, and lightweight tools.
个人创作的网页小游戏、交互玩具、实验性项目与轻量工具合集。

This repository contains small-scale creative and technical experiments.
Each project is self-contained and stored in its own directory.
本仓库收录个人的小型创意与技术实验项目。

## Projects ｜ 子项目

### 1. Aeterna Memoria

**Type ｜ 类型：** Interactive Web Toy / 交互式网页玩具

**Technology ｜ 技术：** WebGL (Three.js), MediaPipe Hands

A browser-based interactive visual experience controlled by hand gestures.
基于浏览器的手势控制交互视觉体验项目。

Users open their hand to summon a visual shield and intercept flying objects.
The objective is to capture exactly the required number of objects in each round to complete a hidden numeric sequence.
用户张开手掌召唤视觉法阵并拦截飞行物体。
每一轮需精确捕捉指定数量的目标，以完成隐藏的数字序列。

**Main features ｜ 主要功能：**

* Real-time hand tracking via webcam
  实时摄像头手部追踪
* WebGL-rendered visual effects
  基于 WebGL 的视觉效果渲染
* Sequence-based interaction logic
  序列驱动的交互逻辑
* Single-file deployment (`index.html`)
  单文件部署结构

**Live Demo ｜ 在线体验：**

[https://aeterna-memoria-for-sara.netlify.app/](https://aeterna-memoria-for-sara.netlify.app/)

**Location ｜ 项目路径：**

```
Aeterna-Memoria/
```

**Usage ｜ 使用方式**

Example ｜ 示例：

```
Aeterna-Memoria/index.html
```

Open the file in a modern browser and allow camera access when prompted.
使用现代浏览器打开文件，并在提示时允许摄像头权限。
